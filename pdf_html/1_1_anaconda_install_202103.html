# 설치 순서
01 'anaconda download' 입력 후, 검색한다.
Individual Edition - Anaconda를 선택하여 이동합니다.
(* 해당 링크명은 달라질 수 있습니다.)

02 ANACONDA 웹 사이트로 이동한다.
(그림 1-2 아나콘다 사이트) 2020/09 시점.
'Download' 다운로드를 누르면 다운로드 할수 있는 화면의 아래쪽으로 이동합니다.


자신의 컴퓨터 버전에 맞춰 다운로드를 실시합니다.
현재 나오는 컴퓨터의 경우 일반적으로 원도우의 경우 64bit가 대부분입니다.
'64bit'버전을 다운로드 합니다.
' 64-Bit Graphical Installer (466 MB)'

03 프로그램을 다운로드를 받는다.

선택을 하면 자동으로 다운로드가 진행됩니다.

04 해당 폴더로 이동 후, 프로그램을 실행시킨다.


05 다운로드 한 프로그램을 마우스로 클릭해서 실행합니다.


06 설치를 진행합니다.




'Next' 선택 후, 다음 화면에서 'Install'을 선택 후, 설치를 진행합니다.



검색에 'ana'라고 치면 Anaconda Prompt가 검색된다. 이를 통해 실행이 가능하다.


* 최신 버전의 경우, 일부 프로그램이 돌아가지 않는 현상이 있어,  시간이 조금 지나서 안정화된 버전을 쓰기도 합니다. 2020년 09월 시점 Python 3.8 버전을 사용하고 있습니다.
* 2021년 3월 기점 최신 버전은 conda 4.9.2, python 3.8.5 버전을 사용하고 있습니다.
